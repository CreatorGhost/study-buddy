### Q22.
Explain the concept of "Alternate Key" in a Relational Database Management System with an appropriate example.

**Answer:**

**Solution:** An **Alternate Key** is a candidate key that is **not selected as the Primary Key**. In a table, there may be multiple columns or combinations of columns that can uniquely identify each record — these are called **Candidate Keys**. The one chosen as the main identifier is the **Primary Key**, and all remaining candidate keys are called **Alternate Keys**.

**Example:**

| StudentID | RollNo | Name | Email |
|---|---|---|---|
| S001 | 101 | Aryan | aryan@mail.com |
| S002 | 102 | Priya | priya@mail.com |

Here, `StudentID`, `RollNo`, and `Email` can all uniquely identify a student — they are all candidate keys. If `StudentID` is chosen as the Primary Key, then `RollNo` and `Email` become **Alternate Keys**.

|---|---|---|

### Q27 (a).
Write the definition of a Python function named `LongLines()` which reads the contents of a text file named `'LINES.TXT'` and displays those lines from the file which have **at least 10 words** in it.

For example, if the content of `'LINES.TXT'` is:
```
Once upon a time, there was a woodcutter
He lived in a little house in a beautiful, green wood.
One day, he was merrily chopping some wood.
He saw a little girl skipping through the woods, whistling happily.
The girl was followed by a big gray wolf.
```
Then the function should display:
```
He lived in a little house in a beautiful, green wood.
He saw a little girl skipping through the woods, whistling happily.
```

**Answer:**

```python
def LongLines():
    with open('LINES.TXT', 'r') as file:
        lines = file.readlines()
        for line in lines:
            words = line.split()
            if len(words) >= 10:
                print(line.strip())
```

**Solution:** The function opens `'LINES.TXT'` in read mode and reads all lines. For each line, it splits the line into words using `split()` (which splits on whitespace). If the number of words is 10 or more, it prints the line (using `strip()` to remove the trailing newline character).

Verification:
- `"Once upon a time, there was a woodcutter"` → 8 words → not printed
- `"He lived in a little house in a beautiful, green wood."` → 11 words → **printed**

### Q29.
Write a function `EOReplace()` in Python, which accepts a list `L` of numbers. Thereafter, it **increments all even numbers by 1** and **decrements all odd numbers by 1**.

Output will be:

**Answer:**

```python
def EOReplace(L):
    for i in range(len(L)):
        if L[i] % 2 == 0:
            L[i] += 1    # Even number: increment by 1
        else:
            L[i] -= 1    # Odd number: decrement by 1
    return L

# Example usage:
L = [10, 20, 30, 40, 35, 55]
print(EOReplace(L))  # Output: [11, 21, 31, 41, 34, 54]
```

**Solution:** The function iterates through each element using its index. For each element:
- If even (`% 2 == 0`): add 1 → e.g., 10 → 11, 20 → 21, 30 → 31, 40 → 41
- If odd: subtract 1 → e.g., 35 → 34, 55 → 54

Since lists are mutable, modifications are made in-place.
