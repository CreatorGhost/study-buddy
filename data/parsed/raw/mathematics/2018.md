### Q2.
Find the magnitude of each of the two vectors $\vec{a}$ and $\vec{b}$, having the same magnitude such that the angle between them is $60°$ and their scalar product is $\dfrac{9}{2}$.

**Answer:** $|\vec{a}| = |\vec{b}| = 3$

**Solution:**

Given: $|\vec{a}| = |\vec{b}|$, $\theta = 60°$, and $\vec{a} \cdot \vec{b} = \dfrac{9}{2}$

$$\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$$

$$\frac{9}{2} = |\vec{a}||\vec{a}|\cos 60°$$

$$\frac{9}{2} = |\vec{a}|^2 \times \frac{1}{2}$$

$$|\vec{a}|^2 = 9$$

$$|\vec{a}| = 3 = |\vec{b}|$$

### Q4.
Find the value of $\tan^{-1}\!\sqrt{3} - \cot^{-1}\!\left(-\sqrt{3}\right)$.

**Answer:** $-\dfrac{\pi}{2}$

**Solution:**

Let $\tan^{-1}\!\left(\sqrt{3}\right) - \cot^{-1}\!\left(-\sqrt{3}\right) = k$ (say)

Using the identity $\cot^{-1}(-x) = \pi - \cot^{-1}(x)$:

$$k = \tan^{-1}\!\left(\sqrt{3}\right) - \left(\pi - \cot^{-1}\!\left(\sqrt{3}\right)\right)$$

$$= \frac{\pi}{3} - \left(\pi - \frac{\pi}{6}\right)$$

$$= \frac{\pi}{3} - \pi + \frac{\pi}{6}$$

$$= \frac{\pi}{2} - \pi = -\frac{\pi}{2}$$

The total cost $C(x)$ associated with the production of $x$ units of an item is given by:
$$C(x) = 0.005x^3 - 0.02x^2 + 30x + 5000$$

$$= 0.135 - 0.12 + 30$$

### Q6.
Differentiate $\tan^{-1}\!\left(\dfrac{1 + \cos x}{\sin x}\right)$ with respect to $x$.

**Answer:** $\dfrac{dy}{dx} = -\dfrac{1}{2}$

**Solution:**

Let $y = \tan^{-1}\!\left(\dfrac{1 + \cos x}{\sin x}\right)$

$$y = \tan^{-1}\!\left(\frac{2\cos^2\frac{x}{2}}{2\sin\frac{x}{2}\cos\frac{x}{2}}\right)$$

$$= \tan^{-1}\!\left(\cot\frac{x}{2}\right)$$

$$= \tan^{-1}\!\left(\tan\!\left(\frac{\pi}{2} - \frac{x}{2}\right)\right)$$

$$\therefore\; y = \frac{\pi}{2} - \frac{x}{2}$$

$$\frac{dy}{dx} = \frac{d}{dx}\!\left(\frac{\pi}{2} - \frac{x}{2}\right) = -\frac{1}{2}$$

### Q7.
Given $A = \begin{bmatrix} 2 & -3 \\ -4 & 7 \end{bmatrix}$, compute $A^{-1}$ and show that $2A^{-1} = 9I - A$.

**Answer:** $A^{-1} = \dfrac{1}{2}\begin{bmatrix} 7 & 3 \\ 4 & 2 \end{bmatrix}$

**Solution:**

$$A = \begin{bmatrix} 2 & -3 \\ -4 & 7 \end{bmatrix}$$

$$|A| = 14 - 12 = 2$$

$$A_{11} = 7,\quad A_{12} = 4,\quad A_{21} = 3,\quad A_{22} = 2$$

$$\text{adj}(A) = \begin{bmatrix} A_{11} & A_{22} \\ A_{21} & A_{22} \end{bmatrix}^T = \begin{bmatrix} 7 & 4 \\ 3 & 2 \end{bmatrix}^T = \begin{bmatrix} 7 & 3 \\ 4 & 2 \end{bmatrix}$$

$$\therefore\; A^{-1} = \frac{1}{|A|}\,\text{adj}(A) = \frac{1}{2}\begin{bmatrix} 7 & 3 \\ 4 & 2 \end{bmatrix}$$

**Verification:**

$$\text{L.H.S.} = 2A^{-1} = \begin{bmatrix} 7 & 3 \\ 4 & 2 \end{bmatrix}$$

$$\text{R.H.S.} = 9I - A = \begin{bmatrix} 9 & 0 \\ 0 & 9 \end{bmatrix} - \begin{bmatrix} 2 & -3 \\ -4 & 7 \end{bmatrix} = \begin{bmatrix} 7 & 3 \\ 4 & 2 \end{bmatrix}$$

$$\text{L.H.S.} = \text{R.H.S.} \qquad \blacksquare$$

### Q8.
Prove that $3\sin^{-1} x = \sin^{-1}(3x - 4x^3)$, $x \in \left[-\dfrac{1}{2}, \dfrac{1}{2}\right]$.

$$\therefore\; \sin 3\theta = 3x - 4x^3$$

$$\Rightarrow\; 3\theta = \sin^{-1}(3x - 4x^3)$$

$$\Rightarrow\; 3\sin^{-1} x = \sin^{-1}(3x - 4x^3) \qquad \blacksquare$$

### Q10.
If $\theta$ is the angle between two vectors $\hat{i} - 2\hat{j} + 3\hat{k}$ and $3\hat{i} - 2\hat{j} + \hat{k}$, find $\sin\theta$.

**Answer:** $\sin\theta = \dfrac{2\sqrt{6}}{7}$

**Solution:**

$$\vec{a} = \hat{i} - 2\hat{j} + 3\hat{k}, \quad \vec{b} = 3\hat{i} - 2\hat{j} + \hat{k}$$

$$\vec{a} \times \vec{b} = \begin{vmatrix} \hat{i} & \hat{j} & \hat{k} \\ 1 & -2 & 3 \\ 3 & -2 & 1 \end{vmatrix} = (4)\hat{i} + 8\hat{j} + 4\hat{k}$$

$$|\vec{a} \times \vec{b}| = \sqrt{(4)^2 + (8)^2 + (4)^2} = \sqrt{16 + 64 + 16} = \sqrt{96} = 4\sqrt{6}$$

$$|\vec{a}| = \sqrt{1 + 4 + 9} = \sqrt{14}, \quad |\vec{b}| = \sqrt{9 + 4 + 1} = \sqrt{14}$$

$$\sin\theta = \frac{|\vec{a} \times \vec{b}|}{|\vec{a}||\vec{b}|} = \frac{4\sqrt{6}}{\sqrt{14} \times \sqrt{14}} = \frac{4\sqrt{6}}{14} = \frac{2\sqrt{6}}{7}$$

### Q11.
Find the differential equation representing the family of curves $y = ae^{bx+5}$, where $a$ and $b$ are arbitrary constants.

**Answer:** $y\dfrac{d^2y}{dx^2} - \left(\dfrac{dy}{dx}\right)^2 = 0$

**Solution:**

$$y = ae^{bx} \times e^5 = \alpha e^{bx}, \quad \text{where } \alpha = e^5 a$$

Differentiating with respect to $x$:

$$\frac{dy}{dx} = \alpha b e^{bx} = by$$

$$\therefore\; \frac{\frac{dy}{dx}}{y} = b$$

Differentiating again with respect to $x$:

$$\frac{y\dfrac{d^2y}{dx^2} - \dfrac{dy}{dx} \cdot \dfrac{dy}{dx}}{y^2} = 0$$

$$y\frac{d^2y}{dx^2} - \left(\frac{dy}{dx}\right)^2 = 0$$

### Q12.
Evaluate:

**Answer:** $\tan x + C$

$$I = \int \frac{\cos 2x + 2\sin^2 x}{\cos^2 x}\,dx$$

### Q13.
If $y = \sin(\sin x)$, prove that:
$$\frac{d^2y}{dx^2} + \tan x\,\frac{dy}{dx} + y\cos^2 x = 0$$

**Solution:**

$$y = \sin(\sin x) \quad \ldots(1)$$

$$\frac{dy}{dx} = \cos(\sin x) \cdot \cos x \;\Rightarrow\; \frac{\dfrac{dy}{dx}}{\cos x} = \cos(\sin x) \quad \ldots(2)$$

$$\frac{d^2y}{dx^2} = -\cos(\sin x)\cdot\sin x - \cos x\cdot\sin(\sin x)\cdot\cos x \quad \ldots(3)$$

Substituting $(1)$ and $(2)$ into $(3)$:

$$\frac{d^2y}{dx^2} = -\left(\frac{\dfrac{dy}{dx}}{\cos x}\right)\sin x - y\cos^2 x$$

$$\frac{d^2y}{dx^2} = -\frac{dy}{dx}\tan x - y\cos^2 x$$

$$\therefore\; \frac{d^2y}{dx^2} + \tan x\,\frac{dy}{dx} + y\cos^2 x = 0 \qquad \blacksquare$$

### Q19.
Find:
$$\int \frac{2\cos x}{(1 - \sin x)(1 + \sin^2 x)}\,dx$$

**Answer:** $\dfrac{1}{2}\log\left|\dfrac{\sin^2 x + 1}{(1-\sin x)^2}\right| + \tan^{-1}(\sin x) + C$

$$\frac{2}{(1-t)(1+t^2)} = \frac{A}{1-t} + \frac{Bt+C}{t^2+1}$$

$$2 = A(t^2+1) + (Bt+C)(1-t)$$

$$2 = (A-B)t^2 + (B-C)t + (A+C)$$

Comparing coefficients:

$$A - B = 0,\quad B - C = 0,\quad A + C = 2$$

$$\Rightarrow A = 1,\; B = 1,\; C = 1$$

$$\therefore\; I = \int\!\left(\frac{1}{1-t} + \frac{2t}{2(t^2+1)} + \frac{1}{t^2+1}\right)dt$$

$$= -\log|1-t| + \frac{1}{2}\log|t^2+1| + \tan^{-1}(t) + C$$

$$= \frac{1}{2}\log\left|\frac{t^2+1}{(1-t)^2}\right| + \tan^{-1}(t) + C$$

$$= \frac{1}{2}\log\left|\frac{\sin^2 x + 1}{(1-\sin x)^2}\right| + \tan^{-1}(\sin x) + C$$

### Q20.
Suppose a girl throws a die. If she gets 1 or 2, she tosses a coin three times and notes the number of tails. If she gets 3, 4, 5 or 6, she tosses a coin once and notes whether a 'head' or 'tail' is obtained. If she obtained exactly one 'tail', what is the probability that she threw 3, 4, 5 or 6 with the die?

**Answer:** $\dfrac{8}{11}$

**Solution:**

Let $A$ = event that girl gets 1 or 2: $P(A) = \dfrac{2}{6} = \dfrac{1}{3}$

Let $B$ = event that girl gets 3, 4, 5 or 6: $P(B) = \dfrac{4}{6} = \dfrac{2}{3}$

$$P\!\left(\frac{T}{A}\right) = \text{Probability of exactly one tail given 1 or 2} = \frac{3}{8}$$

$$P\!\left(\frac{T}{B}\right) = \text{Probability of exactly one tail given 3,4,5,6} = \frac{1}{2}$$

By Bayes' theorem:

$$P\!\left(\frac{B}{T}\right) = \frac{P(B)\cdot P\!\left(\dfrac{T}{B}\right)}{P(A)\cdot P\!\left(\dfrac{T}{A}\right) + P(B)\cdot P\!\left(\dfrac{T}{B}\right)}$$

$$= \frac{\dfrac{2}{3}\times\dfrac{1}{2}}{\dfrac{1}{3}\times\dfrac{3}{8} + \dfrac{2}{3}\times\dfrac{1}{2}} = \frac{\dfrac{1}{3}}{\dfrac{1}{8}+\dfrac{1}{3}} = \frac{\dfrac{1}{3}}{\dfrac{11}{24}} = \frac{8}{11}$$

### Q22.
An open tank with a square base and vertical sides is to be constructed from a metal sheet so as to hold a given quantity of water. Show that the cost of material will be least when depth of the tank is half of its width. If the cost is to be borne by nearby settled lower income families, for whom water will be provided, what kind of value is hidden in this question?

**Solution:**

Let the length, width, and height of the open tank be $x$, $x$, and $y$ units respectively.

Volume: $V = x^2 y$ (constant)

Total surface area: $S = x^2 + 4xy$

$$S = x^2 + 4xy = x^2 + \frac{4V}{x}$$

$$\frac{dS}{dx} = 2x - \frac{4V}{x^2}, \qquad \frac{d^2S}{dx^2} = 2 + \frac{8V}{x^3}$$

Setting $\dfrac{dS}{dx} = 0$:

$$2x - \frac{4V}{x^2} = 0 \;\Rightarrow\; 2x^3 = 4V = 4x^2y \;\Rightarrow\; x = 2y$$

Since $\dfrac{d^2S}{dx^2} = 2 + \dfrac{8V}{x^3} > 0$ for all $x$, $S$ is minimum when $x = 2y$.

### Q26.
Let $A = \{x \in \mathbb{Z} : 0 \leq x \leq 12\}$. Show that $R = \{(a,b) : a, b \in A,\, |a-b| \text{ is divisible by } 4\}$ is an equivalence relation. Find the set of all elements related to 1. Also write the equivalence class $[2]$.

**Answer:** Elements related to $1$: $\{1, 5, 9\}$; $[2] = \{2, 6, 10\}$

**Solution:**

$R = \{(a,b): |a-b| \text{ is a multiple of } 4\}$, where $a,b \in A = \{0,1,2,\ldots,12\}$

**Reflexivity:** For any $a \in A$:

$$|a - a| = 0 = 4 \times 0, \text{ which is a multiple of } 4$$

$$\Rightarrow (a, a) \in R$$

So $R$ is reflexive. ✓

**Symmetry:** Let $(a,b) \in R$. Then:

$$|a-b| = 4\lambda \text{ for some } \lambda \in \mathbb{N}$$

$$|b-a| = 4\lambda \quad [\because |a-b| = |b-a|]$$

$$\Rightarrow (b,a) \in R$$

So $R$ is symmetric. ✓

**Transitivity:** Let $(a,b) \in R$ and $(b,c) \in R$. Then:

$$|a-b| = 4\lambda \text{ and } |b-c| = 4\mu \text{ for some } \lambda,\mu \in \mathbb{N}$$

$$a - b = \pm 4\lambda \text{ and } b - c = \pm 4\mu$$

$$a - c = \pm 4\lambda \pm 4\mu = \pm 4(\lambda \pm \mu)$$

$$\Rightarrow |a-c| \text{ is a multiple of } 4 \;\Rightarrow\; (a,c) \in R$$

So $R$ is transitive. ✓

Hence, $R$ is an equivalence relation.

**Elements related to 1:** Let $x \in A$ such that $(x,1) \in R$. Then:

$$|x-1| \text{ is a multiple of } 4 \;\Rightarrow\; |x-1| \in \{0,4,8,12\}$$

$$x - 1 \in \{0,4,8,12\} \;\Rightarrow\; x \in \{1,5,9,13\}$$

Since $13 \notin A$: Set of elements related to $1$ is $\{1, 5, 9\}$, i.e., $[1] = \{1, 5, 9\}$

$$[2] = \{2, 6, 10\}$$
