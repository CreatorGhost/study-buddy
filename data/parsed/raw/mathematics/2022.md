### Q2.
In a parallelogram PQRS, $\overrightarrow{PQ} = 3\hat{i} - 2\hat{j} + 2\hat{k}$ and $\overrightarrow{PS} = -\hat{i} - 2\hat{k}$. Find $\overrightarrow{PR}$ and $\overrightarrow{QS}$.

**Answer:** $|\overrightarrow{PR}| = 2\sqrt{2}$, $|\overrightarrow{QS}| = 6$

**Solution:**

$$\overrightarrow{PR} = \overrightarrow{PQ} + \overrightarrow{PS} = (3\hat{i} - 2\hat{j} + 2\hat{k}) + (-\hat{i} - 2\hat{k}) = 2\hat{i} - 2\hat{j}$$

$$|\overrightarrow{PR}| = \sqrt{(2)^2 + (-2)^2 + 0^2} = \sqrt{8} = 2\sqrt{2}$$

$$\overrightarrow{QS} = \overrightarrow{PS} - \overrightarrow{PQ} = (-\hat{i} - 2\hat{k}) - (3\hat{i} - 2\hat{j} + 2\hat{k}) = -4\hat{i} + 2\hat{j} - 4\hat{k}$$

$$|\overrightarrow{QS}| = \sqrt{(-4)^2 + (2)^2 + (-4)^2} = \sqrt{36} = 6$$

- $|\overrightarrow{PR}| = 2\sqrt{2}$
- $|\overrightarrow{QS}| = 6$

### Q3(a).
If $\dfrac{d}{dx}[F(x)] = \dfrac{\sec^4 x}{\cosec^4 x}$ and $F\!\left(\dfrac{\pi}{4}\right) = \dfrac{\pi}{4}$, then find $F(x)$.

**Answer:** $F(x) = \tan^3 x + \dfrac{\pi}{4} - 1$

**Solution:**

**Simplifying the derivative:**

$$\frac{\sec^4 x}{\cosec^4 x} = \frac{\left(\dfrac{1}{\cos^4 x}\right)}{\left(\dfrac{1}{\sin^4 x}\right)} = \left(\frac{\sin x}{\cos x}\right)^4 = \tan^4 x$$

**Integration:** $F(x) = \displaystyle\int \tan^4 x \, dx$

Use the identity:
$$\tan^4 x = \sec^4 x - 2\sec^2 x + 1$$

So:
$$F(x) = \int(\sec^4 x - 2\sec^2 x + 1)\,dx$$

Integrate term by term:
$$F(x) = \int \sec^4 x\,dx - 2\int \sec^2 x\,dx + \int 1\,dx$$

- The integral of $\sec^2 x$ is $\tan x$.
- The integral of $1$ is $x$.

Finally:
$$F(x) = \tan^3 x + C$$

**Applying the initial condition** $F\!\left(\dfrac{\pi}{4}\right) = \dfrac{\pi}{4}$:

$$\tan^3\!\left(\frac{\pi}{4}\right) + C = \frac{\pi}{4}$$

$$1 + C = \frac{\pi}{4} \implies C = \frac{\pi}{4} - 1$$

$$\boxed{F(x) = \tan^3 x + \frac{\pi}{4} - 1}$$

**Answer:** $-\dfrac{1}{2(\log x)^2} + \dfrac{1}{(\log x)^3} + C$

**Solution:**

Then the integral becomes:

$$I = \int \frac{u - 3}{u^4}\,e^u\,du$$

Substitute back:

$$I = -\frac{1}{2(\log x)^2} + \frac{1}{(\log x)^3} + C$$

$$\boxed{-\frac{1}{2(\log x)^2} + \frac{1}{(\log x)^3} + C}$$

### Q4.
Let A and B be two events such that $P(A) = \dfrac{5}{8}$, $P(B) = \dfrac{1}{2}$, and $P(A/B) = \dfrac{3}{4}$. Find the value of $P(B/A)$.

**Answer:** $\dfrac{3}{5}$

**Solution:**

We know that:
$$P(A \mid B) = \frac{P(A \cap B)}{P(B)}$$

Substituting the given values:
$$\frac{3}{4} = \frac{P(A \cap B)}{\dfrac{1}{2}}$$

Solving for $P(A \cap B)$:
$$P(A \cap B) = \frac{3}{4} \times \frac{1}{2} = \frac{3}{8}$$

Now, using the definition of conditional probability:
$$P(B \mid A) = \frac{P(A \cap B)}{P(A)} = \frac{\dfrac{3}{8}}{\dfrac{5}{8}} = \frac{3}{5}$$

$$\boxed{P(B/A) = \frac{3}{5}}$$

### Q5.
Two balls are drawn at random from a bag containing 2 red balls and 3 blue balls, without replacement. Let the variable X denote the number of blue balls in the drawn set. Write the distribution of X.

**Answer:** See distribution table below.

**Solution:**

**Total Number of Balls:**
The bag contains 2 red balls and 3 blue balls. Hence, the total number of balls is: $n = 2 + 3 = 5$.

**Possible Values of X:**
$X$ can take the values $0, 1,$ or $2$, representing the number of blue balls drawn.

**Probability Calculation for Each Value of X:**

$$P(X = 0) = \frac{\dbinom{3}{2}}{\dbinom{5}{2}} = \frac{3}{10}$$

$$P(X = 1) = \frac{\dbinom{2}{1} \times \dbinom{3}{1}}{\dbinom{5}{2}} = \frac{6}{10} = \frac{3}{5}$$

**Case 3: $X = 2$ (Both balls are blue):**

$$P(X = 2) = \frac{\dbinom{3}{2}}{\dbinom{5}{2}} = \frac{3}{10}$$

**Final Probability Distribution:**

| $X$ | $P(X)$ |
|-----|--------|
| 0 | $\dfrac{3}{10}$ |
| 1 | $\dfrac{3}{5}$ |
| 2 | $\dfrac{1}{10}$ |

### Q6.
Find the value of $\lambda$ for which the distance between the points $(2, 1, \lambda)$ from plane $3x + 5y + 4z = 11$ is equal to $2\sqrt{2}$ units.

**Answer:** $\lambda = \pm 1$

**Solution:**

The distance $D$ from a point $(x_1, y_1, z_1)$ to a plane $ax + by + cz + d = 0$ is given by:

$$D = \frac{|ax_1 + by_1 + cz_1 + d|}{\sqrt{a^2 + b^2 + c^2}}$$

Given plane equation: $3x + 5y + 4z = 11$, rewritten as $3x + 5y + 4z - 11 = 0$.

Given point: $(2, 1, \lambda)$. Given distance: $D = 2\sqrt{2}$.

Substituting the values into the distance formula:

$$2\sqrt{2} = \frac{|3(2) + 5(1) + 4(\lambda) - 11|}{\sqrt{3^2 + 5^2 + 4^2}}$$

$$2\sqrt{2} = \frac{|6 + 5 + 4\lambda - 11|}{\sqrt{9 + 25 + 16}} = \frac{|4\lambda|}{\sqrt{50}} = \frac{2|2\lambda|}{\sqrt{50}}$$

$$\Rightarrow |2\lambda| = 4 \Rightarrow \lambda = \pm 1$$

$$\boxed{\lambda = \pm 1}$$

### Q7(b).
The two adjacent sides of a parallelogram are represented by $2\hat{i} - 4\hat{j} - 5\hat{k}$ and $2\hat{i} + 2\hat{j} + 3\hat{k}$. Find the unit vectors parallel to its diagonals. Using the diagonal vectors, find the area of the parallelogram also.

**Answer:** Unit vector along $\vec{A}+\vec{B}$ is $\dfrac{2\hat{i}-\hat{j}-\hat{k}}{\sqrt{6}}$; Unit vector along $\vec{A}-\vec{B}$ is $-\dfrac{3\hat{j}+4\hat{k}}{5}$; Area $= 2\sqrt{221}$

**Solution:**

Let $\vec{A} = 2\hat{i} - 4\hat{j} - 5\hat{k}$ and $\vec{B} = 2\hat{i} + 2\hat{j} + 3\hat{k}$.

The diagonals of the parallelogram are $\vec{A} + \vec{B}$ and $\vec{A} - \vec{B}$.

$$\vec{A} + \vec{B} = (2+2)\hat{i} + (-4+2)\hat{j} + (-5+3)\hat{k} = 4\hat{i} - 2\hat{j} - 2\hat{k}$$

**Unit vector parallel to $\vec{A} + \vec{B}$:**

$$\frac{\vec{A}+\vec{B}}{|\vec{A}+\vec{B}|} = \frac{4\hat{i}-2\hat{j}-2\hat{k}}{\sqrt{(4)^2+(-2)^2+(-2)^2}} = \frac{4\hat{i}-2\hat{j}-2\hat{k}}{\sqrt{24}} = \frac{4\hat{i}-2\hat{j}-2\hat{k}}{2\sqrt{6}} = \frac{2\hat{i}-\hat{j}-\hat{k}}{\sqrt{6}}$$

Similarly:
$$\vec{A} - \vec{B} = (2-2)\hat{i} + (-4-2)\hat{j} + (-5-3)\hat{k} = -6\hat{j} - 8\hat{k}$$

**Unit vector parallel to $\vec{A} - \vec{B}$:**

$$\frac{\vec{A}-\vec{B}}{|\vec{A}-\vec{B}|} = \frac{-6\hat{j}-8\hat{k}}{\sqrt{(-6)^2+(-8)^2}} = \frac{-6\hat{j}-8\hat{k}}{\sqrt{100}} = \frac{-6\hat{j}-8\hat{k}}{10} = -\frac{3\hat{j}+4\hat{k}}{5}$$

**Area of the parallelogram** is $|\vec{A} \times \vec{B}|$:

$$\vec{A} \times \vec{B} = \begin{vmatrix} \hat{i} & \hat{j} & \hat{k} \\ 2 & -4 & -5 \\ 2 & 2 & 3 \end{vmatrix} = \hat{i}(12+10) - \hat{j}(6+10) + \hat{k}(4+8) = 22\hat{i} - 16\hat{j} + 12\hat{k}$$

$$\text{Area} = |\vec{A} \times \vec{B}| = \sqrt{22^2 + (-16)^2 + 12^2} = \sqrt{484 + 256 + 144} = \sqrt{884} = 2\sqrt{221}$$

### Q9(a).
Find $\displaystyle\int \frac{dx}{\sqrt{x} + \sqrt[3]{x}}$

**Solution:**

Let $I = \displaystyle\int \frac{dx}{\sqrt{x} + \sqrt[3]{x}}$

Let $u = \sqrt{x}$ and $v = \sqrt[3]{x}$, so $x = u^2 = v^3$.

Substitute $u = \sqrt{x}$ and differentiate:
$$du = \frac{1}{2\sqrt{x}}\,dx = \frac{1}{2u}\,dx \implies dx = 2u\,du$$

The integral becomes:

$$I = \int \frac{2u\,du}{u + \dfrac{v}{u^2}} = 2\int \frac{u^2\,du}{u^3 + v^3}$$

Use the identity $u^3 + v^3 = (u+v)(u^2 - uv + v^2)$:

$$I = 2\int \frac{du}{u^2 - uv + v^2}$$

The integral can now be solved using standard methods.

### Q10.
Find the particular solution of the differential equation:

$$x\frac{dy}{dx} + x\cos^2\!\left(\frac{y}{x}\right) = y$$

given that when $x = 1$, $y = \dfrac{\pi}{4}$.

**Solution:**

The given equation is:
$$x\frac{dy}{dx} + x\cos^2\!\left(\frac{y}{x}\right) = y$$

Rearrange to standard form:
$$\frac{dy}{dx} + \cos^2\!\left(\frac{y}{x}\right) = \frac{y}{x}$$

Let $u = \dfrac{y}{x}$, so $y = ux$ and $\dfrac{dy}{dx} = u + x\dfrac{du}{dx}$.

Substituting:
$$u + x\frac{du}{dx} + \cos^2(u) = u$$

$$x\frac{du}{dx} = -\cos^2(u)$$

**Separate variables:**
$$\frac{du}{\cos^2(u)} = -\frac{dx}{x}$$

**Integrate both sides:**
$$\int \sec^2(u)\,du = -\int \frac{dx}{x}$$

$$\tan(u) = -\ln|x| + C$$

**Substitute back** $u = \dfrac{y}{x}$:

$$\tan\!\left(\frac{y}{x}\right) = -\ln|x| + C$$

**Apply the initial condition** $x = 1$, $y = \dfrac{\pi}{4}$:

$$\tan\!\left(\frac{\pi/4}{1}\right) = -\ln|1| + C$$

$$1 = 0 + C \implies C = 1$$

**Particular solution:**

$$\boxed{\tan\!\left(\frac{y}{x}\right) = -\ln|x| + 1}$$

### Q11(b).
Using integration, find the area of the region bounded by the lines $x - y + 1 = 0$, $x = -2$, $x = 3$, and the x-axis.

**Solution:**

### Q12.
A card from a pack of 52 playing cards is lost. From the remaining cards, 2 cards are drawn at random without replacement, and are found to be both aces. Find the probability that the lost card being an ace.

**Answer:** $\dfrac{1}{25}$

**Solution:**

- Total number of cards in the deck $= 52$
- Number of aces in the deck $= 4$
- Since one card is lost, the total number of remaining cards $= 51$

There are two possible cases:
1. The lost card is an ace.
2. The lost card is not an ace.

**Case 1: Lost card is an ace.**

In this case, the number of aces left $= 3$.

$$P(\text{2 aces drawn} \mid \text{lost is ace}) = \frac{\dbinom{3}{2}}{\dbinom{51}{2}} = \frac{3}{1275}$$

**Case 2: Lost card is not an ace.**

In this case, the number of aces left $= 4$.

$$P(\text{2 aces drawn} \mid \text{lost is not ace}) = \frac{\dbinom{4}{2}}{\dbinom{51}{2}} = \frac{6}{1275}$$

**Prior probabilities:**
$$P(\text{lost is ace}) = \frac{4}{52}, \quad P(\text{lost is not ace}) = \frac{48}{52}$$

**By Bayes' Theorem:**

$$P(\text{lost is ace} \mid \text{2 aces drawn}) = \frac{\dfrac{3}{1275} \times \dfrac{4}{52}}{\dfrac{3}{1275} \times \dfrac{4}{52} + \dfrac{6}{1275} \times \dfrac{48}{52}}$$

**Simplifying:**

$$= \frac{12}{12 + 288} = \frac{12}{300} = \frac{1}{25}$$

$$\boxed{P(\text{lost card is an ace}) = \frac{1}{25}}$$

### Q13.
Evaluate $\displaystyle\int_0^{\pi} \frac{x}{1 + \sin x}\,dx$

**Answer:** $\dfrac{\pi}{2}\ln 2$

**Solution:**

Let $I = \displaystyle\int_0^{\pi} \frac{x}{1+\sin x}\,dx$

Using the property $\displaystyle\int_0^a f(x)\,dx = \int_0^a f(a-x)\,dx$:

$$I = \int_0^{\pi} \frac{\pi - x}{1 + \sin(\pi - x)}\,dx$$

Since $\sin(\pi - x) = \sin x$:

$$I = \int_0^{\pi} \frac{\pi - x}{1 + \sin x}\,dx$$

**Adding the two integrals:**

$$I + I = \int_0^{\pi} \frac{x + (\pi - x)}{1+\sin x}\,dx = \int_0^{\pi} \frac{\pi}{1+\sin x}\,dx$$

$$2I = \pi \int_0^{\pi} \frac{dx}{1+\sin x}$$

**To solve** $\displaystyle\int_0^{\pi} \frac{dx}{1+\sin x}$, let $u = 1 + \sin x$, so $du = \cos x\,dx$.

$$\int_0^{\pi} \frac{dx}{1+\sin x} = \int_1^2 \frac{du}{u} = \ln\left(\frac{2}{1}\right) = \ln 2$$

Thus:
$$2I = \pi \ln 2 \implies I = \frac{\pi}{2}\ln 2$$

$$\boxed{I = \frac{\pi}{2}\ln 2}$$

### Q14.
Electrical transmission wires which are laid down in winters are stretched tightly to accommodate expansion in summers.

[Diagram: Image of electrical transmission towers with overhead ground wires, insulators, and nearest three phase conductors on grounded metal towers]

Two such wires lie along the following lines:

$$l_1 : \frac{x+1}{3} = \frac{y-3}{-2} = \frac{z+2}{-1}$$

$$l_2 : \frac{x}{-1} = \frac{y-7}{3} = \frac{z+7}{-2}$$

Based on the given information, answer the following questions:

**(i) Are the lines $l_1$ and $l_2$ coplanar? Justify your answer.** [2 marks]

**(ii) Find the point of intersection of the lines $l_1$ and $l_2$.** [2 marks]

---

**Solution — Part (i): Coplanarity of $l_1$ and $l_2$**

The direction ratios of the lines are:
$$\vec{a_1} = (3, -2, -1), \quad \vec{a_2} = (-1, 3, -2)$$

Let the point on line $l_1$ be $P_1(-1, 3, -2)$ and the point on line $l_2$ be $P_2(0, 7, -7)$.

Find the vector $\overrightarrow{P_1P_2}$:
$$\overrightarrow{P_1P_2} = (0-(-1),\; 7-3,\; -7-(-2)) = (1, 4, -5)$$

To check for coplanarity, find the scalar triple product $[\overrightarrow{P_1P_2},\, \vec{a_1},\, \vec{a_2}]$:

$$[\overrightarrow{P_1P_2},\, \vec{a_1},\, \vec{a_2}] = \begin{vmatrix} 1 & 4 & -5 \\ 3 & -2 & -1 \\ -1 & 3 & -2 \end{vmatrix}$$

$$= 1((-2)(-2) - (-1)(3)) - 4((3)(-2) - (-1)(-1)) + (-5)((3)(3) - (-2)(-1))$$

$$= 1(4+3) - 4(-6-1) + (-5)(9-2)$$

$$= 1(5) + 4(-8) + (-5)(-12) = 5 - 32 + 60 = 33 \neq 0$$

Since the scalar triple product is **not zero**, the lines $l_1$ and $l_2$ are **not coplanar**.

---

**Solution — Part (ii): Point of Intersection**

Write the parametric forms:

**For $l_1$:** $\quad x = 3t-1,\quad y = -2t+3,\quad z = -t-2$

**For $l_2$:** $\quad x = -u,\quad y = 3u+7,\quad z = -2u-7$

Equate $x$, $y$, $z$ from both equations:

$$3t - 1 = -u \quad \Rightarrow \quad u = -3t + 1 \tag{1}$$

$$-2t + 3 = 3u + 7 \tag{2}$$

$$-t - 2 = -2u - 7 \tag{3}$$

Substitute $(1)$ into $(2)$:

$$-2t + 3 = 3(-3t+1) + 7 = -9t + 3 + 7 = -9t + 10$$

$$7t = 7 \implies t = 1$$

$$u = -3(1) + 1 = -2$$

Substitute $t = 1$ back into the parametric equations for $l_1$:

$$x = 3(1) - 1 = 2$$
$$y = -2(1) + 3 = 1$$
$$z = -(1) - 2 = -3$$

$$\boxed{\text{Point of intersection} = (2,\; 1,\; -3)}$$
