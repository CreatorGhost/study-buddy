If $\vec{a} = 7\hat{i} + \hat{j} - 4\hat{k}$ and $\vec{b} = 2\hat{i} + 6\hat{j} + 3\hat{k}$, then find the projection of $\vec{a}$ on $\vec{b}$.

### Q5.

Find the differential equation representing the family of curves $v = \dfrac{A}{r} + B$, where $A$ and $B$ are arbitrary constants.

**Answer:** $r^2\dfrac{d^2v}{dr^2} + 2r\dfrac{dv}{dr} = 0$

**Solution:**

$$\frac{dv}{dr} = -\frac{A}{r^2}$$

Differentiating again:

$$\frac{d^2v}{dr^2} = \frac{2A}{r^3}$$

Eliminating $A$: from $\dfrac{dv}{dr} = -\dfrac{A}{r^2}$, we get $A = -r^2\dfrac{dv}{dr}$. Substituting:

$$r^2\frac{d^2v}{dr^2} + 2r\frac{dv}{dr} = 0$$

### Q6.

Find the integrating factor of the differential equation

$$\left(\frac{e^{-2\sqrt{x}}}{\sqrt{x}} - \frac{y}{\sqrt{x}}\right)\frac{dx}{dy} = 1$$

**Answer:** $e^{2\sqrt{x}}$

**Solution:**

Rewriting the equation:

$$\frac{dy}{dx} + \frac{1}{\sqrt{x}}\cdot y = \frac{e^{-2\sqrt{x}}}{\sqrt{x}}$$

This is a linear ODE in $y$, with $P = \dfrac{1}{\sqrt{x}}$.

$$I.F = e^{\int \frac{1}{\sqrt{x}}\,dx} = e^{2\sqrt{x}}$$

### Q8.

If $f(x) = \begin{vmatrix} a & -1 & 0 \\ ax & a & -1 \\ ax^2 & ax & a \end{vmatrix}$, using properties of determinants find the value of $f(2x) - f(x)$.

**Answer:** $ax(3x + 2a)$

**Solution:**

Apply $R_2 \to R_2 - xR_1$ and $R_3 \to R_3 - x^2R_1$:

$$f(x) = \begin{vmatrix} a & -1 & 0 \\ 0 & a+x & -1 \\ 0 & ax+x^2 & a \end{vmatrix}$$

Expanding along the first column:

$$f(x) = a\bigl[(a+x)(a) - (-1)(ax + x^2)\bigr] = a(a^2 + ax + ax + x^2) = a(a+x)^2$$

Therefore:

$$f(2x) - f(x) = a(2x+a)^2 - a(x+a)^2 = a\bigl[(2x+a)^2 - (x+a)^2\bigr]$$

$$= a(2x+a+x+a)(2x+a-x-a) = a(3x+2a)(x) = ax(3x+2a)$$

$$= \frac{1}{6}\log|1-\cos x| + \frac{1}{2}\log|1+\cos x| - \frac{2}{3}\log|1+2\cos x| + C$$

### Q10.

Evaluate: $\displaystyle\int_{-\pi}^{\pi} (\cos ax - \sin bx)^2\, dx$

**Answer:** $2\pi$ (or more precisely $2\pi + \dfrac{1}{2a}\sin 2a\pi - \dfrac{\sin 2b\pi}{2b}$)

**Solution:**

$$I = \int_{-\pi}^{\pi} (\cos ax - \sin bx)^2\,dx = \int_{-\pi}^{\pi}(\cos^2 ax + \sin^2 bx)\,dx - \int_{-\pi}^{\pi} 2\cos ax\sin bx\,dx$$

$$= I_1 - I_2$$

Since $\cos^2 ax + \sin^2 bx$ is an **even** function:

$$I_1 = 2\int_0^{\pi}(\cos^2 ax + \sin^2 bx)\,dx$$

Since $2\cos ax \sin bx$ is an **odd** function:

$$I_2 = 0$$

Therefore:

$$I = I_1 = \int_0^{\pi}(1+\cos 2ax + 1 - \cos 2bx)\,dx$$

$$= \left[2x + \frac{\sin 2ax}{2a} - \frac{\sin 2bx}{2b}\right]_0^{\pi}$$

$$= 2\pi + \frac{1}{2a}\sin 2a\pi - \frac{\sin 2b\pi}{2b} \quad \text{or simply } 2\pi$$

### Q11.

A bag A contains 4 black and 6 red balls and bag B contains 7 black and 3 red balls. A die is thrown. If 1 or 2 appears on it, then bag A is chosen, otherwise bag B. If two balls are drawn at random (without replacement) from the selected bag, find the probability of one of them being red and another black.

**OR**

An unbiased coin is tossed 4 times. Find the mean and variance of the number of heads obtained.

**Answer (Main):** $\dfrac{22}{45}$

**Solution (Main):**

Let $E_1$: selecting bag A, $E_2$: selecting bag B.

$$P(E_1) = \frac{1}{3}, \quad P(E_2) = \frac{2}{3}$$

Let $A$: getting one red and one black ball.

$$P(A|E_1) = \frac{{}^4C_1 \cdot {}^6C_1}{{}^{10}C_2} = \frac{24}{45} = \frac{8}{15}$$

$$P(A|E_2) = \frac{{}^7C_1 \cdot {}^3C_1}{{}^{10}C_2} = \frac{21}{45} = \frac{7}{15}$$

By the law of total probability:

$$P(A) = P(E_1)\cdot P(A|E_1) + P(E_2)\cdot P(A|E_2)$$

$$= \frac{1}{3}\cdot\frac{8}{15} + \frac{2}{3}\cdot\frac{7}{15} = \frac{8}{45} + \frac{14}{45} = \frac{22}{45}$$

---

**Answer (OR):** Mean $= 2$, Variance $= 1$

**Solution (OR):**

For $n = 4$ tosses of an unbiased coin, $p = q = \dfrac{1}{2}$.

| $x$ | $P(x)$ | $xP(x)$ | $x^2P(x)$ |
|-----|---------|---------|----------|
| 0 | $\dfrac{1}{16}$ | $0$ | $0$ |
| 1 | $\dfrac{4}{16}$ | $\dfrac{4}{16}$ | $\dfrac{4}{16}$ |
| 2 | $\dfrac{6}{16}$ | $\dfrac{12}{16}$ | $\dfrac{24}{16}$ |
| 3 | $\dfrac{4}{16}$ | $\dfrac{12}{16}$ | $\dfrac{36}{16}$ |
| 4 | $\dfrac{1}{16}$ | $\dfrac{4}{16}$ | $\dfrac{16}{16}$ |

$$\text{Mean} = \sum x\,P(x) = \frac{32}{16} = 2$$

$$\text{Variance} = \sum x^2 P(x) - \left(\sum x\,P(x)\right)^2 = \frac{80}{16} - 4 = 5 - 4 = 1$$

### Q12.

If $\vec{r} = x\hat{i} + y\hat{j} + z\hat{k}$, find $(\vec{r} \times \hat{i})\cdot(\vec{r} \times \hat{j}) + xy$.

**Answer:** $0$

**Solution:**

$$\vec{r} \times \hat{i} = (x\hat{i}+y\hat{j}+z\hat{k})\times\hat{i} = -y\hat{k} + z\hat{j}$$

$$\vec{r} \times \hat{j} = (x\hat{i}+y\hat{j}+z\hat{k})\times\hat{j} = x\hat{k} - z\hat{i}$$

Therefore:

$$(\vec{r}\times\hat{i})\cdot(\vec{r}\times\hat{j}) = (0\hat{i}+z\hat{j}-y\hat{k})\cdot(-z\hat{i}+0\hat{j}+x\hat{k})$$

$$= 0 + 0 + (-y)(x) = -xy$$

Hence:

$$(\vec{r}\times\hat{i})\cdot(\vec{r}\times\hat{j}) + xy = -xy + xy = 0$$

### Q16.

If $x = a\cos\theta + b\sin\theta$, $y = a\sin\theta - b\cos\theta$, show that $y^2\dfrac{d^2y}{dx^2} - x\dfrac{dy}{dx} + y = 0$.

**Solution:**

$$\frac{dx}{d\theta} = -a\sin\theta + b\cos\theta, \qquad \frac{dy}{d\theta} = a\cos\theta + b\sin\theta$$

$$\frac{dy}{dx} = \frac{dy/d\theta}{dx/d\theta} = \frac{a\cos\theta+b\sin\theta}{-a\sin\theta+b\cos\theta} = -\frac{x}{y}$$

(Since $x = a\cos\theta+b\sin\theta$ and $y = a\sin\theta - b\cos\theta$, and $\dfrac{dy}{dx} = -\dfrac{x}{y}$.)

So:

$$y\frac{dy}{dx} + x = 0$$

Differentiating with respect to $x$:

$$y\frac{d^2y}{dx^2} + \frac{dy}{dx}\cdot\frac{dy}{dx} + 1 = 0$$

Multiplying through by $y$:

$$y^2\frac{d^2y}{dx^2} + \left(\frac{dy}{dx}\right)^2 y + y = 0$$

Since $\dfrac{dy}{dx} = -\dfrac{x}{y}$, we get $y\cdot\left(\dfrac{dy}{dx}\right)^2 = y\cdot\dfrac{x^2}{y^2} = \dfrac{x^2}{y}$. Substituting back:

$$y^2\frac{d^2y}{dx^2} - x\frac{dy}{dx} + y = 0 \qquad \blacksquare$$

### Q20.

Let N denote the set of all natural numbers and R be the relation on $N \times N$ defined by $(a, b)\,R\,(c, d)$ if $ad(b+c) = bc(a+d)$. Show that R is an equivalence relation.

**Solution:**

**Reflexivity:** For all $(a,b) \in N\times N$:

$$ab(b+a) = ba(a+b) \implies (a,b)\,R\,(a,b)$$

∴ R is reflexive. $\quad \cdots(i)$

**Symmetry:** Let $(a,b)\,R\,(c,d)$, so $ad(b+c) = bc(a+d)$.

Then $cb(d+a) = da(c+b)$, which means $(c,d)\,R\,(a,b)$.

∴ R is symmetric. $\quad \cdots(ii)$

**Transitivity:** Let $(a,b)\,R\,(c,d)$ and $(c,d)\,R\,(e,f)$ for $a,b,c,d,e,f \in N$.

$$ad(b+c)=bc(a+d) \implies \frac{b+c}{bc} = \frac{a+d}{ad} \implies \frac{1}{c}+\frac{1}{b} = \frac{1}{d}+\frac{1}{a} \quad \cdots(A)$$

$$cf(d+e)=de(c+f) \implies \frac{1}{e}+\frac{1}{d} = \frac{1}{f}+\frac{1}{c} \quad \cdots(B)$$

Adding $(A)$ and $(B)$:

$$\frac{1}{c}+\frac{1}{b}+\frac{1}{e}+\frac{1}{d} = \frac{1}{d}+\frac{1}{a}+\frac{1}{f}+\frac{1}{c}$$

$$\implies \frac{1}{b}+\frac{1}{e} = \frac{1}{a}+\frac{1}{f} \implies af(b+e) = be(a+f)$$

Hence $(a,b)\,R\,(e,f)$, so R is transitive. $\quad \cdots(iii)$

From $(i)$, $(ii)$ and $(iii)$, **R is an equivalence relation**. $\blacksquare$

### Q21.

Using integration, find the area of the triangle formed by the positive x-axis and the tangent and normal to the circle $x^2 + y^2 = 4$ at $(1, \sqrt{3})$.

**OR**

Evaluate $\displaystyle\int_1^3 (e^{2-3x} + x^2 + 1)\,dx$ as a limit of a sum.

**Answer (Main):** $2\sqrt{3}$ sq. units

**Solution (Main):**

[Diagram: Circle $x^2+y^2=4$ centred at origin with radius 2. Point $P(1,\sqrt{3})$ on the circle. Normal OP passes through the origin. Tangent at P meets the x-axis at Q(4,0).]

**Equation of normal (OP):** $y = \sqrt{3}x$

**Equation of tangent (PQ):** The tangent to $x^2+y^2=4$ at $(1,\sqrt{3})$ is:

$$x + \sqrt{3}y = 4 \implies y = \frac{1}{\sqrt{3}}(4-x)$$

Coordinates of Q: $(4, 0)$

Required area:

$$= \int_0^1 \sqrt{3}x\,dx + \int_1^4 \frac{1}{\sqrt{3}}(4-x)\,dx$$

### Q24.

### Q25.

Find the local maxima and local minima of the function $f(x) = \sin x - \cos x$, $0 < x < 2\pi$. Also find the local maximum and local minimum values.

**Answer:**
- Local maximum at $x = \dfrac{3\pi}{4}$, value $= \sqrt{2}$
- Local minimum at $x = \dfrac{7\pi}{4}$, value $= -\sqrt{2}$

**Solution:**

$$f'(x) = \cos x + \sin x$$

Setting $f'(x) = 0$: $\tan x = -1 \implies x = \dfrac{3\pi}{4},\; \dfrac{7\pi}{4}$

$$f''(x) = -\sin x + \cos x$$

At $x = \dfrac{3\pi}{4}$:

$$f''\!\left(\frac{3\pi}{4}\right) = -\frac{1}{\sqrt{2}} - \frac{1}{\sqrt{2}} = -\sqrt{2} < 0 \implies \text{Local Maxima}$$

$$f\!\left(\frac{3\pi}{4}\right) = \frac{1}{\sqrt{2}}+\frac{1}{\sqrt{2}} = \sqrt{2}$$

At $x = \dfrac{7\pi}{4}$:

$$f''\!\left(\frac{7\pi}{4}\right) = \frac{1}{\sqrt{2}} + \frac{1}{\sqrt{2}} = \sqrt{2} > 0 \implies \text{Local Minima}$$

$$f\!\left(\frac{7\pi}{4}\right) = -\frac{1}{\sqrt{2}}-\frac{1}{\sqrt{2}} = -\sqrt{2}$$
