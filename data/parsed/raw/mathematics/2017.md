### Q1.
If for any $2 \times 2$ square matrix $A$, $A(\text{adj } A) = \begin{bmatrix} 8 & 0 \\ 0 & 8 \end{bmatrix}$, then write the value of $|A|$.

**Answer:** $|A| = 8$

**Solution:**
Given:
$$A(\text{adj } A) = \begin{bmatrix} 8 & 0 \\ 0 & 8 \end{bmatrix}$$

Using the property $A(\text{adj } A) = |A| \cdot I_n$:

$$|A| \cdot I_n = \begin{bmatrix} 8 & 0 \\ 0 & 8 \end{bmatrix}$$

$$|A| \cdot I_n = 8\begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix}$$

$$\Rightarrow |A| = 8$$

### Q2.
Determine the value of $k$ for which the following function is continuous at $x = 3$:

$$f(x) = \begin{cases} \dfrac{(x+3)^2 - 36}{x - 3} & , \ x \neq 3 \\[10pt] k & , \ x = 3 \end{cases}$$

**Answer:** $k = 12$

**Solution:**
$$\lim_{x \to 3} f(x) = \lim_{x \to 3} \frac{(x+3)^2 - 36}{x - 3}$$

$$= \lim_{x \to 3} \frac{(x+3-6)(x+3+6)}{(x-3)}$$

$$= \lim_{x \to 3} \frac{(x-3)(x+9)}{(x-3)} = \lim_{x \to 3}(x + 9) = 12$$

Since $f(x)$ is continuous at $x = 3$:
$$\lim_{x \to 3} f(x) = f(3)$$
$$\Rightarrow k = 12$$

Find the distance between the planes $2x - y + 2z = 5$ and $5x - 2.5y + 5z = 20$.

If $A$ is a skew-symmetric matrix of order 3, then prove that $\det A = 0$.

**Solution:**

Let $A = \begin{bmatrix} 0 & a & b \\ -a & 0 & c \\ -b & -c & 0 \end{bmatrix}$ be a skew-symmetric matrix of order 3.

$$|A| = \begin{vmatrix} 0 & a & b \\ -a & 0 & c \\ -b & -c & 0 \end{vmatrix}$$

$$|A| = -a(0 + bc) + b(ac - 0)$$

$$= -abc + abc = 0 \quad \textbf{Proved}$$

### Q6.
Find the value of $c$ in Rolle's theorem for the function $f(x) = x^3 - 3x$ in $[-\sqrt{3},\ 0]$.

**Answer:** $c = -1$

**Solution:**

$f(x) = x^3 - 3x$

**(i)** $f(x)$ being a polynomial is continuous on $[-\sqrt{3},\ 0]$.

**(ii)** $f(-\sqrt{3}) = -3\sqrt{3} + 3\sqrt{3} = 0$ and $f(0) = 0$, so $f(-\sqrt{3}) = f(0) = 0$.

**(iii)** $f'(x) = 3x^2 - 3$, which exists uniquely on $(-\sqrt{3},\ 0)$.

$\therefore f(x)$ satisfies all conditions of Rolle's theorem.

$\therefore$ There exists at least one $c \in (-\sqrt{3},\ 0)$ where $f'(c) = 0$:

$$3c^2 - 3 = 0 \Rightarrow c^2 = 1 \Rightarrow c = \pm 1$$

Since $c \in (-\sqrt{3},\ 0)$:
$$\Rightarrow c = -1$$

### Q7.
The volume of a cube is increasing at the rate of $9 \text{ cm}^3/\text{s}$. How fast is its surface area increasing when the length of an edge is $10$ cm?

**Answer:** $3.6 \text{ cm}^2/\text{s}$

**Solution:**

Let volume of cube $= V$, side $= l$.

Given: $\dfrac{dV}{dt} = 9 \text{ cm}^3/\text{s}$, $l = 10 \text{ cm}$.

$$\frac{dV}{dt} = \frac{d}{dt}(l^3) = 3l^2\frac{dl}{dt} = 9$$

$$\Rightarrow \frac{dl}{dt} = \frac{3}{l^2} \quad \cdots(1)$$

Now, surface area $A = 6l^2$:

$$\frac{dA}{dt} = \frac{d}{dt}(6l^2) = 12l\frac{dl}{dt} = 12l \times \frac{3}{l^2} = \frac{36}{l}$$

$$= \frac{36}{10} = 3.6 \text{ cm}^2/\text{s}$$

### Q8.
Show that the function $f(x) = x^3 - 3x^2 + 6x - 100$ is increasing on $\mathbb{R}$.

**Solution:**

$$f(x) = x^3 - 3x^2 + 6x - 100$$

$$f'(x) = 3x^2 - 6x + 6$$

$$f'(x) = 3(x^2 - 2x + 2)$$

$$f'(x) = 3\left[(x-1)^2 + 1\right]$$

Since $(x-1)^2 \geq 0$ for all $x \in \mathbb{R}$:

$$f'(x) \geq 3(0 + 1) = 3 > 0 \quad \text{for all } x \in \mathbb{R}$$

$\therefore f(x)$ is increasing on $\mathbb{R}$. $\blacksquare$

### Q9.
The $x$-coordinate of a point on the line joining the points $P(2, 2, 1)$ and $Q(5, 1, -2)$ is $4$. Find its $z$-coordinate.

**Answer:** $z = -1$

**Solution:**

Let $R$ divide $PQ$ in the ratio $k : 1$.

$$R = \left(\frac{5k+2}{k+1},\ \frac{k+2}{k+1},\ \frac{-2k+1}{k+1}\right)$$

Given $x$-coordinate of $R = 4$:

$$\frac{5k+2}{k+1} = 4 \Rightarrow 5k + 2 = 4k + 4 \Rightarrow k = 2$$

$\therefore$ $z$-coordinate:

$$z = \frac{-2(2)+1}{2+1} = \frac{-4+1}{3} = \frac{-3}{3} = -1$$

### Q10.
A die, whose faces are marked $1, 2, 3$ in red and $4, 5, 6$ in green, is tossed. Let $A$ be the event "number obtained is even" and $B$ be the event "number obtained is red". Find if $A$ and $B$ are independent events.

**Answer:** $A$ and $B$ are **not** independent events.

**Solution:**

$$A = \{2, 4, 6\} \Rightarrow P(A) = \frac{3}{6} = \frac{1}{2}$$

$$B = \{1, 2, 3\} \Rightarrow P(B) = \frac{3}{6} = \frac{1}{2}$$

$$A \cap B = \{2\} \Rightarrow P(A \cap B) = \frac{1}{6}$$

Now check:
$$P(A) \cdot P(B) = \frac{1}{2} \times \frac{1}{2} = \frac{1}{4}$$

Since $P(A \cap B) = \dfrac{1}{6} \neq \dfrac{1}{4} = P(A) \cdot P(B)$,

$\therefore$ Events $A$ and $B$ are **not** independent.

### Q15.
If $x^y + y^x = a^b$, then find $\dfrac{dy}{dx}$.

**OR**

If $e^y(x+1) = 1$, then show that $\dfrac{d^2y}{dx^2} = \left(\dfrac{dy}{dx}\right)^2$.

**Solution:**

We have $x^y + y^x = a^b$.

Differentiating w.r.t. $x$:
$$\frac{d}{dx}(x^y) + \frac{d}{dx}(y^x) = 0 \quad \cdots(1)$$

Let $u = x^y$, so $\log u = y\log x$:
$$\frac{1}{u}\frac{du}{dx} = \frac{y}{x} + \log x \cdot \frac{dy}{dx}$$
$$\Rightarrow \frac{d}{dx}(x^y) = x^y\left(\frac{y}{x} + \log x\frac{dy}{dx}\right) \quad \cdots(2)$$

Let $v = y^x$, so $\log v = x\log y$:
$$\frac{1}{v}\frac{dv}{dx} = \frac{x}{y}\frac{dy}{dx} + \log y$$
$$\Rightarrow \frac{d}{dx}(y^x) = y^x\left(\frac{x}{y}\frac{dy}{dx} + \log y\right) \quad \cdots(3)$$

Substituting (2) and (3) in (1):
$$x^y\left(\frac{y}{x} + \log x\frac{dy}{dx}\right) + y^x\left(\frac{x}{y}\frac{dy}{dx} + \log y\right) = 0$$

$$\left(x^y \log x + \frac{xy^{x-1} \cdot x}{1}\right)\frac{dy}{dx} = -\left(yx^{y-1} + y^x\log y\right)$$

$$\therefore \frac{dy}{dx} = -\frac{y^x\log y + yx^{y-1}}{x^y\log x + xy^{x-1}}$$

---

**OR**

Given: $e^y(x+1) = 1$.

Differentiating w.r.t. $x$:
$$e^y(1) + (x+1)e^y\frac{dy}{dx} = 0$$

$$\Rightarrow (x+1)\frac{dy}{dx} + 1 = 0 \quad \cdots(1)$$

Differentiating again w.r.t. $x$:
$$(x+1)\frac{d^2y}{dx^2} + \left(\frac{dy}{dx}\right)\cdot 1 = 0$$

$$\frac{d^2y}{dx^2} = -\frac{\dfrac{dy}{dx}}{(x+1)}$$

From (1): $(x+1) = -\dfrac{1}{\dfrac{dy}{dx}}$, so:

$$\frac{d^2y}{dx^2} = -\frac{dy}{dx} \cdot \frac{dy}{dx}$$

$$\Rightarrow \frac{d^2y}{dx^2} = \left(\frac{dy}{dx}\right)^2 \quad \blacksquare$$

### Q18.
Solve the differential equation $(\tan^{-1}x - y)\,dx = (1+x^2)\,dy$.

**Answer:** $y = \tan^{-1}x - 1 + ce^{-\tan^{-1}x}$

**Solution:**

Rearranging:
$$\frac{dy}{dx} = \frac{\tan^{-1}x - y}{1+x^2}$$

$$\frac{dy}{dx} + \frac{y}{1+x^2} = \frac{\tan^{-1}x}{1+x^2}$$

This is a linear ODE. Integrating factor:

$$I.F. = e^{\int\frac{1}{1+x^2}dx} = e^{\tan^{-1}x}$$

Multiplying:
$$y \cdot e^{\tan^{-1}x} = \int \frac{\tan^{-1}x}{1+x^2} \cdot e^{\tan^{-1}x}\,dx$$

Let $t = \tan^{-1}x$, $dt = \dfrac{dx}{1+x^2}$:

$$= \int t\,e^t\,dt = te^t - e^t + c = e^t(t-1) + c$$

$$\therefore y\,e^{\tan^{-1}x} = e^{\tan^{-1}x}(\tan^{-1}x - 1) + c$$

$$y = \tan^{-1}x - 1 + ce^{-\tan^{-1}x}$$

**$f$ is one-one:**

Let $x_1, x_2 \in \mathbb{R} - \left\{-\dfrac{4}{3}\right\}$ and $f(x_1) = f(x_2)$:

$$\frac{4x_1+3}{3x_1+4} = \frac{4x_2+3}{3x_2+4}$$

$$\Rightarrow 7x_1 = 7x_2 \Rightarrow x_1 = x_2$$

$\therefore f$ is one-one.

**$f$ is onto:**

Let $k \in \mathbb{R} - \left\{\dfrac{4}{3}\right\}$ be any number. Set $f(x) = k$:

$$\frac{4x+3}{3x+4} = k \Rightarrow x = \frac{4k-3}{4-3k}$$

One can verify $\dfrac{4k-3}{4-3k} \neq -\dfrac{4}{3}$ (since that leads to a contradiction $-9 = -16$).

$\therefore f$ is bijective, hence invertible.

**Inverse of $f$:**

Let $f^{-1}(x) = k$, so $f(k) = x$:
$$\frac{4k+3}{3k+4} = x \Rightarrow k = \frac{4x-3}{4-3x}$$

$$\therefore f^{-1}(x) = \frac{4x-3}{4-3x},\quad x \in \mathbb{R}-\left\{-\frac{4}{3}\right\}$$

**OR**

**Commutativity:** $(a,b)*(c,d) = (ac, b+ad)$ but $(c,d)*(a,b) = (ca, d+cb)$.

Since $b+ad \neq d+cb$ in general, $*$ is **not commutative**.

**Associativity:**

$$[(a,b)*(c,d)]*(e,f) = (ac, b+ad)*(e,f) = (ace, b+ad+acf)$$

$$(a,b)*[(c,d)*(e,f)] = (a,b)*(ce, d+cf) = (ace, b+a(d+cf)) = (ace, b+ad+acf)$$

Since both are equal, $*$ is **associative**.

**(i) Identity element:**

$$(a,b)*(e,f) = (a,b) \Rightarrow (ae, b+af) = (a,b)$$

### Q26.
Show that the surface area of a closed cuboid with square base and given volume is minimum, when it is a cube.

**Solution:**

Let each side of the square base be $x$ and height be $h$.

Volume: $V = x^2 h \Rightarrow h = \dfrac{V}{x^2}$

Surface area:
$$S = 4hx + 2x^2 = 4\!\left(\frac{V}{x^2}\right)\!x + 2x^2 = \frac{4V}{x} + 2x^2$$

Differentiating w.r.t. $x$:
$$\frac{dS}{dx} = -\frac{4V}{x^2} + 4x \qquad \text{and} \qquad \frac{d^2S}{dx^2} = \frac{8V}{x^3} + 4$$

Setting $\dfrac{dS}{dx} = 0$:

$$4x = \frac{4V}{x^2} \Rightarrow x^3 = V \Rightarrow x = V^{1/3}$$

At $x = V^{1/3}$: $\dfrac{d^2S}{dx^2} = \dfrac{8V}{V} + 4 = 12 > 0$.

$\therefore S$ is minimum when $x = V^{1/3}$.

And the corresponding height:
$$h = \frac{V}{x^2} = \frac{V}{V^{2/3}} = V^{1/3} = x$$

Since $h = x$, all sides are equal $\Rightarrow$ it is a **cube**. $\blacksquare$
